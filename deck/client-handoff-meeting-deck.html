OPERATING MODEL // 03

You Do Not Operate The Mac Mini

DAILY USE

Telegram is the interface.

Send text or voice notes. Use one topic per operating front. Use Mission Control for system-wide decisions.

INFRASTRUCTURE

The Mac mini runs quietly.

It hosts OpenClaw, memory, browser control, Mission Control, and local services. It should stay powered and online.

TAILSCALE // 05

Secure Access When The Mac Mini Is Not Beside You

01

Install it on your devices.

Computer first. Phone optional. This lets approved devices see the Mac mini securely.

02

03

Telegram is still the daily interface. Tailscale is the secure network layer.

REMOTE SUPPORT // 06

Who Uses What

CLIENT

Uses Telegram.

  • Mission Control topic
  • Dedicated front topics
  • Text and voice notes
  • No terminal for normal use

OPERATOR

Uses Tailscale.

  • Health checks
  • Updates
  • Remote troubleshooting
  • Handoff validation

SCREEN SHARING

Only when useful.

  • AnyDesk or macOS sharing
  • For visual support
  • Not required for daily use
  • Not a replacement for Tailscale

BROWSER CONTROL // 08

When There Is No API, Use The Web Product

DEDICATED AGENT BROWSER

The agent does not use your personal browser.

Approved accounts are signed into the agent browser profile. This keeps automation separate from normal personal browsing.

API EXISTS

Use the API.

Best for reliable, repeatable workflows.

NO NORMAL API

Use the browser.

Good for consumer or partner-restricted tools.

PASSWORD RULE

Do not send passwords in chat. Sign in manually inside the agent browser.
